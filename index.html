Hello Gemba!

A sample repository, just for you

Congratulations on your first Gemba-connected Git repository!

This is a real web page that is hosted on GitHub, just like your real web and mobile apps. When you deliver assets through Gemba, you directly improve the app.

Try it out:

Have you mastered the art of delivering assets?

  1. Grab the checkmark.png image to the left and make it  green  using Sketch or Photoshop.
    (or use Preview: Tools → Adjust Color… → Tint)
  2. Drop the green checkmark into Gemba.
    Open Gemba
  3. Press “Deliver Changes” and reload this page.

See how the green checkmark now compliments you on your asset-integration mastery? You’ve just updated an asset directly in a web page!
